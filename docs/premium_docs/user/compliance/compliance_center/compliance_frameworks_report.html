    • 



{{< /history >}}


With the compliance frameworks report, you can see all the compliance frameworks in a group. Each row of the report shows:


    

  • Framework name.
    • 

  • Associated projects.
    • 



The default framework for the group has a default badge.


View the compliance frameworks report


To view the compliance frameworks report:


    

  1. On the top bar, select Search or go to and find your project or group.
    2. 

  2. Select Secure > Compliance center.
    3. 

  3. On the page, select the Frameworks tab.
    4. 



Create a new compliance framework


Prerequisites:


    

  • You must be an administrator or have the Owner role for the group.
    • 



To create a new compliance framework from the compliance frameworks report:


    

  1. On the top bar, select Search or go to and find your group.
    2. 

  2. Select Secure > Compliance center.
    3. 

  3. On the page, select the Frameworks tab.
    4. 

  4. Select the New framework.
    5. 

  5. Select Create blank framework.
    6. 

  6. Select the Add framework to create compliance framework.
    7. 



Edit a compliance framework


Prerequisites:


    

  • You must be an administrator or have the Owner role for the group.
    • 



To edit a compliance framework from the compliance frameworks report:


    

  1. On the top bar, select Search or go to and find your group.
    2. 

  2. Select Secure > Compliance center.
    3. 

  3. On the page, select the Frameworks tab.
    4. 

  4. Hover over framework and select Edit the framework.
    5. 

  5. Select the Save changes to edit compliance framework.
    6. 



Delete a compliance framework


Prerequisites:


    

  • You must be an administrator or have the Owner role for the group.
    • 



To delete a compliance framework from the compliance frameworks report:


    

  1. On the top bar, select Search or go to and find your group.
    2. 

  2. Select Secure > Compliance center.
    3. 

  3. On the page, select the Frameworks tab.
    4. 

  4. Hover over framework and select Edit the framework.
    5. 

  5. Select the Delete framework to delete compliance framework.
    6. 



Set and remove a compliance framework as default

Prerequisites:


    

  • You must be an administrator or have the Owner role for the group.

To set a compliance framework as default] from the compliance frameworks report:


    

  1. On the top bar, select Search or go to and find your group.
    2. 

  2. Select Secure > Compliance center.
    3. 

  3. On the page, select the Frameworks tab.
    4. 

  4. Next to the compliance framework you want to set as default, select {{< icon name="pencil" >}} action.
    5. 

  5. Select the Set as default to set as default.
    6. 



To remove a compliance framework as default from the compliance frameworks report:


    

  1. On the top bar, select Search or go to and find your group.
    2. 

  2. Select Secure > Compliance center.
    3. 

  3. On the page, select the Frameworks tab.
    4. 

  4. Next to the compliance framework that is default, select {{< icon name="pencil" >}} action.
    5. 

  5. Select the Remove as default to remove as default.
    6. 



Export a report of compliance frameworks in a group


{{< history >}}


    

  • Introduced in GitLab 16.11 with a flag named compliance_frameworks_report_csv_export. Disabled by default.
    • 

  • Generally available in GitLab 17.1. Feature flag compliance_frameworks_report_csv_export removed.
    • 



{{< /history >}}


Exports the contents of a compliance frameworks report in a group. Reports are truncated at 15 MB to avoid a large email attachment.


Prerequisites:


    

  • You must be an administrator or have the Owner role for the group.
    • 



To export the standards adherence report for projects in a group:


    

  1. On the top bar, select Search or go to and find your group.
    2. 

  2. Select Secure > Compliance center.
    3. 

  3. In the upper-right corner, select Export.
    4. 

  4. Select Export framework report.
    5. 



A report is compiled and delivered to your email inbox as an attachment.
